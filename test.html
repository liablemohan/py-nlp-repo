  • श्लोकसंख्या : 001
    वागर्थाविव संपृक्तौ वागर्थप्रतिपत्तये।
    जगतः पितरौ वन्दे पार्वतीपरमेश्वरौ॥
  • श्लोकसंख्या : 002
    क्व सूर्यप्रभवो वंशः क्व चाल्पविषया मतिः।
    तितीर्षुर्दुस्तरं मोहादुडुपेनास्मि सागरम्॥
  • श्लोकसंख्या : 003
    मन्दः कवियशःप्रार्थी गमिष्याम्युपहास्यताम्।
    प्रांशुलभ्ये फले लोभादुद्बाहुरिव वामनः॥
  • श्लोकसंख्या : 004
    अथ वा कृतवाग्द्वारे वंशेऽस्मिन्पूर्वसूरिभिः।
    मणौ वज्रसमुत्कीर्णे सूत्रस्येवास्ति मे गतिः॥
  • श्लोकसंख्या : 005
    सोऽहमाजन्मशुद्धानामाफलोदयकर्मणाम्।
    आसमुद्रक्षितीशानामानाकरथवर्त्मनाम्॥
  • श्लोकसंख्या : 006
    यथाविधिहुताग्नीनां यथाकामार्चितार्थिनाम्।
    यथापराधदण्डानां यथाकालप्रबोधिनाम्॥
  • श्लोकसंख्या : 007
    त्यागाय संभृतार्थानां सत्याय मितभाषिणाम्।
    यशसे विजिगीषूणां प्रजायै गृहमेधिनाम्॥
  • श्लोकसंख्या : 008
    शैशवेऽभ्यस्तविद्यानां यौवने विषयैषिणाम्।
    वार्द्धके मुनिवृत्तीनां योगेनान्ते तनुत्यजाम्॥
  • श्लोकसंख्या : 009
    रघूणामन्वयं वक्ष्ये तनुवाग्विभवोऽपि सन्।
    तद्गुणैः कर्णमागत्य चापलाय प्रचोदितः॥
  • श्लोकसंख्या : 010
    तं सन्तः श्रोतुमर्हन्ति सदसद्व्यक्तिहेतवः।
    हेम्नः संलक्ष्यते ह्यग्नौ विशुद्धिः श्यामिकाऽपि वा॥
  • श्लोकसंख्या : 011
    वैवस्वतो मनुर्नाम माननीयो मनीषिणाम्।
    आसीन्महीक्षितामाद्यः प्रणवश्छन्दसामिव॥
  • श्लोकसंख्या : 012
    तदन्वये शुद्धिमति प्रसूतः शुद्धिमत्तरः।
    दिलीप इति राजेन्दुरिन्दुः क्षीरनिधाविव॥
  • श्लोकसंख्या : 013
    व्यूढोरस्को वृषस्कन्धः शालप्रांशुर्महाभुजः।
    आत्मकर्मक्षमं देहं क्षात्रो धर्म इवाश्रितः॥
  • श्लोकसंख्या : 014
    सर्वातिरिक्तसारेण सर्वतेजोभिभाविना।
    स्थितः सर्वोन्नतेनोर्वीं क्रान्त्वा मेरुरिवात्मना॥
  • श्लोकसंख्या : 015
    आकारसदृशप्रज्ञः प्रज्ञया सदृशागमः।
    आगमैः सदृशारम्भ आरम्भसदृशोदयः॥
  • श्लोकसंख्या : 016
    भीमकान्तैर्नृपगुणैः स बभूवोपजीविनाम्।
    अधृष्यश्चाभिगम्यश्च यादोरत्नैरिवार्णवः॥
  • श्लोकसंख्या : 017
    रेखामात्रमपि क्षुण्णादा मनोर्वर्त्मनः परम्।
    न व्यतीयुः प्रजास्तस्य नियन्तुर्नेमिवृत्तयः॥
  • श्लोकसंख्या : 018
    प्रजानामेव भूत्यर्थं स ताभ्यो बलिमग्रहीत्।
    सहस्रगुणमुत्स्रष्टुमादत्ते हि रसं रविः॥
  • श्लोकसंख्या : 019
    सेनापरिच्छदस्तस्य द्वयमेवार्थसाधनम्।
    शास्त्रेष्वकुण्ठिता बुद्धिर्मौर्वी धनुषि चातता॥
  • श्लोकसंख्या : 020
    तस्य संवृतमन्त्रस्य गूढाकारेङ्गितस्य च।
    फलानुमेयाः प्रारम्भाः संस्काराः प्राक्तना इव॥
  • श्लोकसंख्या : 021
    जुगोपात्मानमत्रस्तो भेजे धर्ममनातुरः।
    अगृध्नुराददे सोऽर्थमसक्तः सुखमन्वभूत्॥
  • श्लोकसंख्या : 022
    ज्ञाने मौनं क्षमा शक्तौ त्यागे श्लाघाविपर्ययः।
    गुणा गुणानुबन्धित्वात्तस्य सप्रसवा इव॥
  • श्लोकसंख्या : 023
    अनाकृष्टस्य विषयैर्विद्यानां पारदृश्वनः।
    तस्य धर्मरतेरासीद्वृद्धत्वं जरसा विना॥
  • श्लोकसंख्या : 024
    प्रजानां विनयाधानाद्रक्षणाद्भरणादपि।
    स पिता पितरस्तासां केवलं जन्महेतवः॥
  • श्लोकसंख्या : 025
    स्थित्यै दण्डयतो दण्ड्यान्परिणेतुः प्रसूतये।
    अप्यर्थकामौ तस्यास्तां धर्म एव मनीषिणः॥
  • श्लोकसंख्या : 026
    दुदोह गां स यज्ञाय सस्याय मघवा दिवम्।
    सम्पद्विनियमयेनोभौ दधतुर्भुवनद्वयम्॥
  • श्लोकसंख्या : 027
    न किलानुययुस्तस्य राजानो रक्षितुर्यशः।
    व्यावृत्ता यत्परस्वेभ्यः श्रुतौ तस्करता स्थिता॥
  • श्लोकसंख्या : 028
    द्वेष्योऽपि सम्मतः शिष्टस्तस्यार्तस्य यथौषधम्।
    त्याज्यो दुष्टः प्रियोऽप्यासीदङ्गुलीवोरगक्षता॥
  • श्लोकसंख्या : 029
    तं वेधा विदधे नूनं महाभूतसमाधिना।
    तथा हि सर्वे तस्यासन्परार्थैकफला गुणाः॥
  • श्लोकसंख्या : 030
    स वेलावप्रवलयां परिखीकृतसागराम्।
    अनन्यशासनामुर्वीं शशासैकपुरीमिव॥
  • श्लोकसंख्या : 031
    तस्य दाक्षिण्यरूढेन नाम्ना मगधवंशजा।
    पत्नी सुदक्षिणेत्यासीदध्वरस्येव दक्षिणा॥
  • श्लोकसंख्या : 032
    कलत्रवन्तमात्मानमवरोधे महत्यपि।
    तया मेने मनस्विन्या लक्ष्म्या च वसुधाधिपः॥
  • श्लोकसंख्या : 033
    तस्यामात्मानुरूपायामात्मजन्मसमुत्सुकः।
    विलम्बितफलैः कालं स निनाय मनोरथैः॥
  • श्लोकसंख्या : 034
    सन्तानार्थाय विधये स्वभुजादवतारिता।
    तेन धूर्जगतो गुर्वी सचिवेषु निचिक्षिपे॥
  • श्लोकसंख्या : 035
    अथाभ्यर्च्य विधातारं प्रयतौ पुत्रकाम्यया।
    तौ दम्पती वसिष्ठस्य गुरोर्जग्मतुराश्रमम्॥
  • श्लोकसंख्या : 036
    स्निग्धगम्भीरनिर्घोषमेकं स्यन्दनमास्थितौ।
    प्रावृषेण्यं पयोवाहं विद्युदैरावताविव॥
  • श्लोकसंख्या : 037
    मा भूदाश्रमपीडेति परिमेयपुरःसरौ।
    अनुभावविशेषात्तु सेनापरिवृताविव॥
  • श्लोकसंख्या : 038
    सेव्यमानौ सुखस्पर्शैः शालनिर्यासगन्धिभिः।
    पुष्परेणूत्किरैर्वातैराधूतवनराजिभिः॥
  • श्लोकसंख्या : 039
    मनोभिरामाः शृण्वन्तौ रथनेमिस्वनोन्मुखैः।
    षड्जसंवादिनीः केका द्विधा भिन्नाः शिखण्डिभिः॥
  • श्लोकसंख्या : 040
    परस्पराक्षिसादृश्यमदूरोज्झितवर्त्मसु।
    मृगद्वन्द्वेषु पश्यन्तौ स्यन्दनाबद्धदृष्टिषु॥
  • श्लोकसंख्या : 041
    श्रेणीबन्धाद्वितन्वद्भिरस्तम्भां तोरणस्रजम्।
    सारसैः कलनिर्ह्रादैः क्वचिदुन्नमिताननौ॥
  • श्लोकसंख्या : 042
    पवनस्यानुकूलत्वात्प्रार्थनासिद्धिशंसिनः।
    रजोभिस्तुरगोत्कीर्णैरस्पृष्टालकवेष्टनौ॥
  • श्लोकसंख्या : 043
    सरसीष्वरविन्दानां वीचिविक्षोभशीतलम्।
    आमोदमुपजिघ्रन्तौ स्वनिःश्वासानुकारिणम्॥
  • श्लोकसंख्या : 044
    ग्रामेष्वात्मविसृष्टेषु यूपचिह्नेषु यज्वनाम्।
    अमोघाः प्रतिगृह्णन्तावर्घ्यानुपदमाशिषः॥
  • श्लोकसंख्या : 045
    हैयङ्गवीनमादाय घोषवृद्धानुपस्थितान्।
    नामधेयानि पृच्छन्तौ वन्यानां मार्गशाखिनाम्॥
  • श्लोकसंख्या : 046
    काऽप्यभिख्या तयोरासीद्व्रजतोः शुद्धवेषयोः।
    हिमनिर्मुक्तयोर्योगे चित्राचन्द्रमसोरिव॥
  • श्लोकसंख्या : 047
    तत्तद्भूमिपतिः पत्न्यै दर्शयन्प्रियदर्शनः।
    अपि लङ्घितमध्वानं बुबुधे न बुधोपमः॥
  • श्लोकसंख्या : 048
    स दुष्प्रापयशाः प्रापदाश्रमं श्रान्तवाहनः।
    सायं संयमिनस्तस्य महर्षेर्महिषीसखः॥
  • श्लोकसंख्या : 049
    वनान्तरादुपावृत्तैः समित्कुशफलाहरैः।
    पूर्यमाणमदृश्याग्निप्रत्युद्यातैस्तपस्विभिः॥
  • श्लोकसंख्या : 050
    आकीर्णमृषिपत्नीनामुटजद्वाररोधिभिः।
    अपत्यैरिव नीवारभागधेयोचितैर्मृगैः॥
  • श्लोकसंख्या : 051
    सेकान्ते मुनिकन्याभिस्तत्क्षणोज्झितवृक्षकम्।
    विश्वासाय विहङ्गानामालवालाम्बुपायिनाम्॥
  • श्लोकसंख्या : 052
    आतपात्ययसंक्षिप्तनीवारासु निषादिभिः।
    मृगैर्वर्तितरोमन्थमुटजाङ्गनभूमिषु॥
  • श्लोकसंख्या : 053
    अभ्युत्थिताग्निपिशुनैरतिथीनाश्रमोन्मुखान्।
    पुनानं पवनोद्धूतैर्धूमैराहुतिगन्धिभिः॥
  • श्लोकसंख्या : 054
    अथ यन्तारमादिश्य धुर्यान्विश्रामयेति सः।
    तामवारोहयत्पत्नीं रथादवततार च॥
  • श्लोकसंख्या : 055
    तस्मै सभ्याः सभार्याय गोप्त्रे गुप्ततमेन्द्रियाः।
    अर्हणामर्हते चक्रुर्मुनयो नयचक्षुषे॥
  • श्लोकसंख्या : 056
    विधेः सायन्तनस्यान्ते स ददर्श तपोनिधिम्।
    अन्वासितमरुन्धत्या स्वाहयेव हविर्भुजम्॥
  • श्लोकसंख्या : 057
    तयोर्जगृहतुः पादान्राजा राज्ञी च मागधी।
    तौ गुरुर्गुरुपत्नी च प्रीत्या प्रतिननन्दतुः॥
  • श्लोकसंख्या : 058
    तमातिथ्यक्रियाशान्तरथक्षोभपरिश्रमम्।
    पप्रच्छ कुशलं राज्ये राज्याश्रममुनिं मुनिः॥
  • श्लोकसंख्या : 059
    अथाथर्वनिधेस्तस्य विजितारिपुरः पुरः।
    अर्थ्यामर्थपतिर्वाचमाददे वदतां वरः॥
  • श्लोकसंख्या : 060
    उपपन्नं ननु शिवं सप्तस्वङ्गेषु यस्य मे।
    दैवीनां मानुषीणां च प्रतिहर्ता त्वमापदाम्॥
  • श्लोकसंख्या : 061
    तव मन्त्रकृतो मन्त्रैर्दूरात्प्रशमितारिभिः।
    प्रत्यादिश्यन्त इव मे दृष्टलक्ष्यभिदः शराः॥
  • श्लोकसंख्या : 062
    हविरावर्जितं होतस्त्वया विधिवदग्निषु।
    वृष्टिर्भवति सस्यानामवग्रहविशोषिणाम्॥
  • श्लोकसंख्या : 063
    पुरुषायुष्यजीविन्यो निरातङ्का निरीतयः।
    यन्मदीयाः प्रजास्तस्य हेतुस्त्वद्ब्रह्मवर्चसम्॥
  • श्लोकसंख्या : 064
    त्वयैवं चिन्त्यमानस्य गुरुणा ब्रह्मयोनिना।
    सानुबन्धाः कथं न स्युः संपदो मे निरापदः॥
  • श्लोकसंख्या : 065
    किन्तु वध्वां तवैतस्यामदृष्टसदृशप्रजम्।
    न मामवति सद्वीपा रत्नसूरपि मेदिनी॥
  • श्लोकसंख्या : 066
    नूनं मत्तः परं वंश्याः पिण्डविच्छेददर्शिनः।
    न प्रकामभुजः श्राद्धे स्वधासंग्रहतत्पराः॥
  • श्लोकसंख्या : 067
    मत्परं दुर्लभं मत्वा नूनमावर्जितं मया।
    पयः पूर्वैः स्वनिःश्वासैः कवोष्णमुपभुज्यते॥
  • श्लोकसंख्या : 068
    सोऽहमिज्याविशुद्धात्मा प्रजालोपनिमीलितः।
    प्रकाशश्चाप्रकाशश्च लोकालोक इवाचलः॥
  • श्लोकसंख्या : 069
    लोकान्तरसुखं पुण्यं तपोदानसमुद्भवम्।
    संततिः शुद्धवंश्या हि परत्रेह च शर्मणे॥
  • श्लोकसंख्या : 070
    तया हीनं विधातर्मां कथं पश्यन्न दूयसे।
    सिक्तं स्वयमिव स्नेहाद्वन्ध्यमाश्रमवृक्षकम्॥
  • श्लोकसंख्या : 071
    असह्यपीडं भगवन्नृणमन्त्यमवेहि मे।
    अरुन्तुदमिवालानमनिर्वाणस्य दन्तिनः॥
  • श्लोकसंख्या : 072
    तस्मान्मुच्ये यथा तात संविधातुं तथार्हसि।
    इक्ष्वाकूणां दुरापेऽर्थे त्वदधीना हि सिद्धयः॥
  • श्लोकसंख्या : 073
    इति विज्ञापितो राज्ञा ध्यानस्तिमितलोचनः।
    क्षणमात्रमृषिस्तस्थौ सुप्तमीन इव ह्रदः॥
  • श्लोकसंख्या : 074
    सोऽपश्यत्प्रणिधानेन सन्ततेः स्तम्भकारणम्।
    भावितात्मा भुवो भर्तुरथैनं प्रत्यबोधयत्॥
  • श्लोकसंख्या : 075
    पुरा शक्रमुपस्थाय तवोर्वीं प्रति यास्यतः।
    आसीत्कल्पतरुच्छायामाश्रिता सुरभिः पथि॥
  • श्लोकसंख्या : 076
    धर्मलोपभयाद्राज्ञीमृतुस्नातामिमां स्मरन्।
    प्रदक्षिणक्रियार्हायां तस्यां त्वं साधु नाचरः॥
  • श्लोकसंख्या : 077
    अवजानासि मां यस्मादतस्ते न भविष्यति।
    मत्प्रसूतिमनाराध्य प्रजेति त्वां शशाप सा॥
  • श्लोकसंख्या : 078
    स शापो न त्वया राजन्न च सारथिना श्रुतः।
    नदत्याकाशगङ्गायाः स्रोतस्युद्दामदिग्गजे॥
  • श्लोकसंख्या : 079
    ईप्सितं तदवज्ञानाद्विद्धि सार्गलमात्मनः।
    प्रतिबध्नाति हि श्रेयः पूज्यपूजाव्यतिक्रमः॥
  • श्लोकसंख्या : 080
    हविषे दीर्घसत्रस्य सा चेदानीं प्रचेतसः।
    भुजङ्गपिहितद्वारं पातालमधितिष्ठति॥
  • श्लोकसंख्या : 081
    सुतां तदीयां सुरभेः कृत्वा प्रतिनिधिं शुचिः।
    आराधय सपत्नीकः प्रीता कामदुघा हि सा॥
  • श्लोकसंख्या : 082
    इति वादिन एवास्य होतुराहुतिसाधनम्।
    अनिन्द्या नन्दिनी नाम धेनुराववृते वनात्॥
  • श्लोकसंख्या : 083
    ललाटोदयमाभुग्नं पल्लवस्निग्धपाटला।
    बिभ्रती श्वेतरोमाङ्कं सन्ध्येव शशिनं नवम्॥
  • श्लोकसंख्या : 084
    भुवं कोष्णेन कुण्डोध्नी मेध्येनावभृथादपि।
    प्रस्रवेनाभिवर्षन्ती वत्सालोकप्रवर्तिना॥
  • श्लोकसंख्या : 085
    रजःकणैः खुरोद्धूतैः स्पृशद्भिर्गात्रमन्तिकात्।
    तीर्थाभिषेकजां शुद्धिमादधाना महीक्षितः॥
  • श्लोकसंख्या : 086
    तां पुण्यदर्शनां दृष्ट्वा निमित्तज्ञस्तपोनिधिः।
    याज्यमाशंसितावन्ध्यप्रार्थनं पुनरब्रवीत्॥
  • श्लोकसंख्या : 087
    अदूरवर्तिनीं सिद्धिं राजन् विगणयात्मनः।
    उपस्थितेयं कल्याणी नाम्नि कीर्तित एव यत्॥
  • श्लोकसंख्या : 088
    वन्यवृत्तिरिमां शश्वदात्मानुगमनेन गाम्।
    विद्यामभ्यसनेनेव प्रसादयितुमर्हसि॥
  • श्लोकसंख्या : 089
    प्रस्थितायां प्रतिष्ठेथाः स्थितायां स्थितिमाचरेः।
    निषण्णायां निषीदास्यां पीताम्भसि पिबेरपः॥
  • श्लोकसंख्या : 090
    वधूर्भक्तिमती चैनामर्चितामा तपोवनात्।
    प्रयता प्रातरन्वेतु सायं प्रत्युद्व्रजेदपि॥
  • श्लोकसंख्या : 091
    इत्याप्रसादादस्यास्त्वं परिचर्यापरो भव।
    अविघ्नमस्तु ते स्थेयाः पितेव धुरि पुत्रिणाम्॥
  • श्लोकसंख्या : 092
    तथेति प्रतिजग्राह प्रीतिमान्सपरिग्रहः।
    आदेशं देशकालज्ञः शिष्यः शासितुरानतः॥
  • श्लोकसंख्या : 093
    अथ प्रदोषे दोषज्ञः संवेशाय विशांपतिम्।
    सूनुः सूनृतवाक्स्रष्टुर्विससर्जोर्जितश्रियम्॥
  • श्लोकसंख्या : 094
    सत्यामपि तपःसिद्धौ नियमापेक्षया मुनिः।
    कल्पवित्कल्पयामास वन्यामेवास्य संविधाम्॥
  • श्लोकसंख्या : 095
    निर्दिष्टां कुलपतिना स पर्णशाला-मध्यास्य प्रयतपरिग्रहद्वितीयः।
    तच्छिष्याध्ययननिवेदितावसानां संविष्टः कुशशयने निशां निनाय॥
